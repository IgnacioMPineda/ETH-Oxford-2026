# ETH-Oxford-2026
Work done and presented at the mathematical Institute at Oxford University for the ETH Oxford hackathon in 2026.

Lancelot is an anonymous, crypto-based platform that protects freelancers from being ghosted by their clients. Each job acts like a "contract" where both parties lock funds as stakes. 

The system has a mathematical-proof-based method to determine whether the work was actually delivered, guaranteeing full security to both freelancer and client, and will then carry out the payment.

Live trust scores can be implemented for both freelancers and clients based on previous interations and following a criteria based on success (the job is carried out satisfactorily) and failure (unsatisfactory job, ghosting,scam. etc.)
